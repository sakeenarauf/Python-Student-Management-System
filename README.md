# Python-Student-Management-System
A simple, menu-driven Python application to manage student data records using files and dictionary structure.
## Features
- Add, View, Search, Update, and Delete Student Records.
- Persistent storage using text files (`students.txt`).
- Computes comprehensive statistics like class average, high/low marks, and top performer.
- ### Design & Methodology
- To keep the application highly understandable and deeply rooted in fundamental concepts, the code strictly avoids complex abstractions or advanced modules (such as `import` statements, `try-except` blocks, or string parsing methods like `split` and `strip`).
